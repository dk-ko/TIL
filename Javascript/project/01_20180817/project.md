## 프로젝트 1 : 계산기

### 1. 문제

- JavaScript로 계산기를 구현하라.
- jQuery로 계산기를 구현하라.

### 2. 필수기능
- 0~9 버튼 : `button` 요소에 클릭 이벤트 핸들러를 적용해 입력을 판별
- +,-,*,/,% 버튼 : 연산 부호를 입력
- Clear 버튼 : 현재 입력된 내용을 초기화
- = 버튼 : 계산결과를표시
- 결과창 : 버튼에서 입력한 내용을 표시하거나 계산 결과를 표시

### 3. 제출
- JavaScript 코드와 jQuery 코드를 각각 KakaoTalk 메신저로 공유한다.
